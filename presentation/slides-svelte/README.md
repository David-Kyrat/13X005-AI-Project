## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
https://github.com/David-Kyrat/svelte-dip-prez-example
```

### 📦️ Install dependencies

```sh
bun i
```

### 💿️ Run the development server

```sh
bun run dev
```
